# How-To Guides

Focused recipes that solve specific problems. Unlike [tutorials](../tutorials/), which teach concepts step by step, guides assume you already understand the basics and need to accomplish a particular task.

## Guides

| Guide | What you'll accomplish |
| ----- | --------------------- |
| [Testing Strategies](testing-strategies.md) | Test automatons at every level — pure functions, runtimes, and integration |
| [Observer Composition](observer-composition.md) | Chain, filter, and combine observers for complex observation pipelines |
| [Upgrading to Decider](upgrading-to-decider.md) | Add command validation to an existing Automaton without breaking changes |
| [Error Handling Patterns](error-handling-patterns.md) | Build Result pipelines with Map, Bind, and MapError |
| [Building Custom Runtimes](building-custom-runtimes.md) | Create your own runtime by wiring Observer and Interpreter |

## Prerequisites

These guides assume familiarity with the [core concepts](../concepts/) and at least [Tutorial 01](../tutorials/01-getting-started.md).
